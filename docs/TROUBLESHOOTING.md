# Troubleshooting

## GUI does not start

- Ensure Tkinter is installed. See `docs/GUI_INSTALLATION.md`.

## No files found

- Verify the input directory path.
- Ensure file extensions are supported.

## Permission errors

- Check read permissions for input directories.
- Check write permissions for output paths.

## Docker issues

- Ensure Docker is installed and running.
- See `docs/DOCKER.md` for usage examples.
